== Description ==

This is a small script I developed to make your WordPress installation completely compatible with the themes developed and distributed by DOTSQR. I developend and use it for my projects. It’s very simple to install, configure and get it running.

Be sure to keep it always enabled to make the theme work flawlessly.

Removing, editing or renaming the plugin files might result in theme configuration errors and issues. If you have any further request, please write me @ hello@dotsqr.co.

What this plugin does:

  • Removes development settings and notifications, making the site suitable for final clients with no technical notions.

== Installation ==

Simply upload and unpack the ‘sqr-theme.zip’ in your plugins directory /wp-content/plugins/ and activate the plugin from WordPress, or install it directly from WordPress plugins repo.
